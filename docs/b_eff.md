아래 설명의 핵심은 이 한 줄이다.

> **RF가 라모어 주파수와 정확히 같으면, 회전좌표계에서 B1은 가만히 서 있는 횡자기장처럼 보인다.
> RF가 라모어 주파수와 다르면, 회전좌표계에서 z축 방향의 가짜 자기장 성분이 생긴 것처럼 보이고, 순자화 M은 B1이 아니라 기울어진 B_eff 둘레로 돈다.**

첨부한 시각화 파일도 같은 구조를 쓰고 있다. 거기서 `Δf = fRF - f0 = 0`이면 B1이 `+x'`에 고정되고, offset이 있으면 `B_eff = B1 x' + (Δω/γ) z`라는 기울어진 축 둘레로 M이 회전한다고 정리되어 있다.

# 1. 가장 기초: MRI에서 양성자는 무엇을 하고 있나

MRI에서 보는 대상은 대체로 수소 원자핵, 즉 **양성자 proton**이다.

양성자는 작은 자석처럼 행동한다. 그래서 외부에서 강한 정자기장 **B0**를 걸면, 양성자의 자기모멘트들이 B0 방향을 기준으로 세차운동을 한다.

보통 MRI 좌표계에서는 이렇게 둔다.

```text
z축  = B0 방향
x-y 평면 = B0에 수직인 평면
```

양성자 하나하나의 자기모멘트는 z축 주변을 빙글빙글 돈다. 이 회전 주파수를 **라모어 주파수**라고 한다.

공식은 다음과 같다.

```text
ω0 = γ B0
f0 = γ B0
```

단, 여기서 표기 주의가 있다.

각주파수로 쓰면,

```text
ω0 = γ B0     단위: rad/s
```

주파수로 쓰면,

```text
f0 = γ̄ B0    단위: Hz
```

여기서 γ̄는 보통 proton의 경우 약

```text
42.58 MHz/T
```

이다.

즉 1.5T MRI에서는 대략,

```text
f0 ≈ 42.58 MHz/T × 1.5 T
   ≈ 63.87 MHz
```

3T MRI에서는,

```text
f0 ≈ 127.7 MHz
```

정도다.

---

# 2. RF pulse란 무엇인가

B0는 z축 방향의 강한 정자기장이다.

그런데 MRI에서는 순자화 M을 z축에 그대로 두면 신호를 잘 얻을 수 없다. 신호를 얻으려면 M을 xy 평면 쪽으로 기울여야 한다.

이를 위해 RF pulse를 건다.

RF pulse는 쉽게 말해,

```text
라모어 주파수 근처에서 회전하는 약한 자기장 B1
```

이다.

B0가 수 T 수준이라면, B1은 보통 μT 수준으로 훨씬 작다. 그런데 B1이 작아도 효과가 큰 이유는, B1이 **양성자들이 이미 돌고 있는 주파수와 맞춰서 같이 돌기 때문**이다.

비유하면 이렇다.

그네를 밀 때, 아무 때나 세게 미는 것보다, 그네의 왕복 주기에 맞춰서 살짝살짝 밀면 에너지가 잘 전달된다.

MRI의 RF pulse도 비슷하다.

```text
양성자의 라모어 회전과 같은 속도로 도는 B1
→ 양성자 입장에서는 계속 같은 방향에서 밀어주는 힘처럼 보임
→ 순자화 M이 점점 기울어짐
```

---

# 3. 실험실 좌표계와 회전좌표계

여기서 가장 중요한 관점 전환이 나온다.

## 3-1. 실험실 좌표계

실험실 좌표계에서는 B0는 z축 방향으로 고정되어 있고, 양성자 자기모멘트들은 xy 평면에서 빠르게 회전한다.

또 RF 자기장 B1도 xy 평면에서 회전한다.

```text
실험실 좌표계:
B0 = z축 방향으로 고정
B1 = xy 평면에서 RF 주파수로 회전
M  = B0 때문에 라모어 주파수로 빠르게 회전
```

이 관점에서는 모든 것이 너무 빠르게 돈다. 1.5T에서 64 MHz, 3T에서 128 MHz 정도로 돈다.

그래서 직관적으로 보기 어렵다.

---

## 3-2. 회전좌표계

그래서 MRI 물리에서는 **회전좌표계 rotating frame**를 쓴다.

회전좌표계는 양성자들의 라모어 회전에 맞춰 같이 도는 좌표계다.

보통 이렇게 쓴다.

```text
x', y' = 라모어 주파수 또는 RF 주파수로 함께 도는 축
z      = 그대로 B0 방향
```

첨부한 시각화 파일에서도 “x′, y′는 f0와 함께 돌고, z축은 B0 방향 그대로”라고 설명한다. 

이 회전좌표계에서 보면 놀라운 일이 생긴다.

RF 주파수가 라모어 주파수와 정확히 같으면,

```text
실험실에서는 빙글빙글 도는 B1
회전좌표계에서는 가만히 서 있는 B1
```

처럼 보인다.

이게 공명 resonance의 핵심이다.

---

# 4. 공명 조건: Δf = 0

RF 주파수를 fRF라고 하자.

라모어 주파수는 f0다.

그러면 RF offset 또는 detuning은 다음처럼 정의한다.

```text
Δf = fRF - f0
```

각주파수로 쓰면,

```text
Δω = ωRF - ω0
```

공명 조건은,

```text
Δf = 0
```

이다.

즉,

```text
fRF = f0
```

이다.

이때 회전좌표계에서는 B1이 +x′ 방향에 고정되어 보인다.

그러면 순자화 M은 어떻게 될까?

M은 B1을 중심축으로 회전한다.

즉,

```text
B0가 z축 방향이라서 M을 z축 주변으로 돌리듯이,
B1은 회전좌표계에서 x′축 방향이므로 M을 x′축 주변으로 돌린다.
```

처음 M이 +z 방향에 있었다면, x′축 둘레로 회전하면서 z에서 y′ 방향으로 내려온다.

예를 들어,

```text
90° RF pulse
→ M이 +z축에서 xy 평면으로 넘어옴

180° RF pulse
→ M이 +z축에서 -z축으로 뒤집힘

360° RF pulse
→ 한 바퀴 돌아 원래 +z축으로 돌아옴
```

첨부 파일도 공명 시 `B1이 +x′에 고정되어 순자화 M을 x′축 둘레로 회전시킨다`고 설명한다. 

---

# 5. RF pulse의 flip angle

B1이 M을 얼마나 기울이는지를 **flip angle**이라고 한다.

공명 상태에서 flip angle은 대략 다음과 같다.

```text
θ = γ B1 t
```

주파수 형태로 쓰면,

```text
θ = 2π f1 t
f1 = γ̄ B1
```

첨부 파일에서도 `θ = 2π f1 t, f1 = γB1`로 정리한다. 

여기서 중요한 점은 이것이다.

```text
flip angle은 B1의 세기와 RF pulse 시간의 곱으로 결정된다.
```

즉,

```text
B1이 세면 짧은 시간에도 많이 기울고,
B1이 약하면 같은 flip을 만들기 위해 더 오래 걸린다.
```

예를 들어 공명 조건에서 90° pulse를 만들고 싶다면,

```text
γ B1 t = π/2
```

가 되도록 B1과 시간을 정하면 된다.

---

# 6. RF offset, Δf란 무엇인가

이제 본론으로 들어가자.

RF offset은 간단히 말해,

```text
RF 주파수가 해당 양성자의 라모어 주파수에서 얼마나 벗어났는가
```

이다.

공식은,

```text
Δf = fRF - f0
```

이다.

예를 들어 어떤 위치의 양성자 라모어 주파수가 63.870 MHz인데, RF를 63.871 MHz로 쏘면,

```text
Δf = +1 kHz
```

이다.

반대로 RF를 63.869 MHz로 쏘면,

```text
Δf = -1 kHz
```

이다.

즉,

```text
Δf > 0 : RF가 양성자보다 빠르다
Δf < 0 : RF가 양성자보다 느리다
Δf = 0 : RF가 양성자와 정확히 맞는다
```

다만 부호는 문헌마다 약간 다르게 정의할 수 있다. 어떤 책은 `Δω = ω0 - ωRF`로 잡기도 한다. 그래서 항상 그 문헌이 어떤 부호 convention을 쓰는지 확인해야 한다.

여기서는 첨부 파일과 같이,

```text
Δf = fRF - f0
```

로 두겠다. 

---

# 7. offset이 있으면 회전좌표계에서 무슨 일이 생기나

공명 조건에서는 회전좌표계에서 B1이 가만히 서 있었다.

그런데 Δf가 0이 아니면 어떻게 될까?

라모어와 함께 도는 좌표계에서 보면 RF B1이 더 이상 완전히 정지해 보이지 않는다.

```text
Δf > 0이면 B1이 회전좌표계에서 앞으로 미끄러져 간다.
Δf < 0이면 B1이 반대 방향으로 미끄러져 간다.
```

첨부 시각화 파일의 상태 설명도 “B1은 x′y′ 평면에서 Δf만큼 드리프트한다”고 되어 있다. 

하지만 MRI 물리에서는 이 현상을 더 편하게 해석하는 방법이 있다.

그게 바로 **effective field**, 즉 **B_eff**다.

---

# 8. B_eff란 무엇인가

회전좌표계에서 offset이 있으면, M은 단순히 B1만 보는 것이 아니다.

M은 마치 다음과 같은 합성 자기장을 보는 것처럼 행동한다.

```text
B_eff = B1 x′ + (Δω / γ) z
```

첨부 파일도 같은 식을 쓴다. 

이 식이 정말 중요하다.

뜻을 풀면 이렇다.

```text
B1 x′
= RF가 만들어낸 횡방향 자기장

(Δω / γ) z
= RF와 라모어 주파수가 어긋난 정도가 만들어낸 z방향 효과
```

즉 offset이 없으면,

```text
Δω = 0

B_eff = B1 x′
```

이 된다.

그러면 M은 순수하게 x′축 주변으로 회전한다.

하지만 offset이 있으면,

```text
B_eff = x′ 방향 성분 + z 방향 성분
```

이 된다.

그래서 B_eff는 x′축에 누워 있지 않고, z축 쪽으로 기울어진다.

---

# 9. 가장 쉬운 그림

공명 상태:

```text
z
↑
│
│    M initially
│    ↑
│    │
└────────→ x′
        B1

B_eff = B1
```

이때 회전축은 x′ 방향이다. 그래서 M은 z에서 y′ 쪽으로 잘 넘어간다.

---

offset이 있는 상태:

```text
z
↑
│      B_eff
│     ↗
│    /
│   /
└────────→ x′
     B1
```

이때 회전축은 x′가 아니라 B_eff다.

그래서 M은 x′축 둘레로 예쁘게 90° 돌아가는 것이 아니라, **기울어진 축 B_eff 둘레로 돈다.**

그 결과,

```text
원래 기대한 flip angle과 실제 M의 방향이 달라진다.
```

---

# 10. 왜 offset이 z방향 성분처럼 보이나

이 부분이 가장 헷갈릴 수 있다.

직관적으로는 이렇게 보면 된다.

회전좌표계를 RF와 함께 돈다고 생각해보자.

그런데 양성자의 실제 라모어 회전 속도가 RF와 다르면, 회전좌표계 안에서도 양성자는 여전히 조금씩 z축 주변으로 도는 것처럼 보인다.

즉,

```text
RF 좌표계에서 봤을 때,
양성자가 RF보다 빠르거나 느리면
상대적인 z축 회전이 남는다.
```

자기모멘트가 z축 주변으로 돈다는 것은, 원래 B0 때문에 생기는 운동이다.

그러므로 회전좌표계에서는 이 남은 상대 회전을 다음처럼 해석할 수 있다.

```text
아, z축 방향으로 남는 자기장이 있는 것처럼 보이는구나.
```

그 가상의 남은 자기장 크기가 바로,

```text
Δω / γ
```

이다.

그래서,

```text
offset = z축 방향 effective field 성분
```

으로 들어간다.

---

# 11. B_eff의 크기

B_eff는 x′ 방향의 B1과 z 방향의 offset 성분의 벡터합이다.

따라서 크기는,

```text
|B_eff| = sqrt( B1² + (Δω/γ)² )
```

주파수 단위로 생각하면 더 쉽다.

B1에 해당하는 nutation frequency를 f1이라고 하자.

```text
f1 = γ̄ B1
```

그러면 effective nutation frequency는,

```text
f_eff = sqrt( f1² + Δf² )
```

이다.

즉 offset이 있으면 M은 B1만 있을 때보다 더 빠른 각속도로 돌 수도 있다.

하지만 중요한 건 속도만이 아니다.

회전축 자체가 기울어진다는 것이 더 중요하다.

---

# 12. B_eff의 기울기

B_eff가 x′축에서 얼마나 z축 쪽으로 기울어지는지는 다음으로 결정된다.

```text
tan α = (Δω/γ) / B1
```

주파수 단위로 쓰면,

```text
tan α = Δf / f1
```

여기서 α는 x′축에서 z축 쪽으로 기울어진 각도다.

즉,

```text
|Δf| << f1
→ B_eff가 거의 x′ 방향
→ 거의 정상적인 flip

|Δf| ≈ f1
→ B_eff가 약 45° 기울어짐
→ flip이 크게 왜곡됨

|Δf| >> f1
→ B_eff가 거의 z축 방향
→ M은 거의 z축 주변으로만 돌고, xy로 잘 넘어오지 않음
```

이게 RF pulse의 주파수 선택성의 핵심이다.

---

# 13. Δf / f1 비율이 핵심이다

offset의 절대값도 중요하지만, 진짜 중요한 것은 보통 이 비율이다.

```text
Δf / f1
```

f1은 B1의 세기에 해당하는 nutation frequency다.

예를 들어 B1이 강해서 f1이 1000 Hz라면, Δf = 100 Hz는 큰 문제가 아닐 수 있다.

```text
Δf / f1 = 0.1
```

B_eff는 거의 x′ 방향이다.

하지만 B1이 약해서 f1이 100 Hz인데 Δf = 100 Hz라면,

```text
Δf / f1 = 1
```

B_eff가 45°나 기울어진다.

따라서 같은 100 Hz offset이라도 B1 세기에 따라 의미가 완전히 달라진다.

---

# 14. 공명에서는 왜 에너지가 잘 전달되나

공명 상태에서는 회전좌표계에서 B1이 정지해 보인다.

M 입장에서는 B1이 계속 같은 방향에서 밀어주는 것처럼 작용한다.

그래서 아주 작은 B1이라도 누적 효과가 생긴다.

```text
B1이 계속 같은 방향으로 M에 토크를 준다
→ M이 일정한 축 주변으로 계속 돈다
→ flip angle이 pulse 시간에 비례해서 증가한다
```

반대로 offset이 크면 B1이 M 입장에서 미끄러져 간다.

그러면 B1이 주는 토크 방향이 계속 달라지고, 효과가 누적되지 않는다.

그 결과 M을 xy 평면으로 잘 눕히지 못한다.

---

# 15. 자전거 바퀴 비유로 이해하기

너가 전에 자전거 바퀴 비유를 계속 써왔으니, 그 방식으로 이어가 보자.

순자화 M을 하나의 자전거 바퀴의 각운동량 벡터라고 보자.

처음에는 M이 z축 위를 향한다.

```text
M = 위쪽을 향한 각운동량
```

B1은 이 각운동량에 토크를 주는 외부 작용이다.

공명 상태에서는 회전좌표계에서 B1이 고정된 방향으로 보인다.

그래서 계속 같은 방향의 토크가 들어온다.

결과적으로 M은 x′축 둘레로 돌아서 z축에서 y′축 쪽으로 넘어간다.

그런데 RF offset이 있으면, 회전좌표계에서도 B1이 완전히 고정되지 않는다. 또는 더 정교하게는, M이 보는 회전축이 B1이 아니라 B_eff가 된다.

그러면 M은 x′축 둘레로 돌아가는 것이 아니라, 기울어진 축 둘레로 돈다.

자전거 바퀴로 말하면,

```text
바퀴의 각운동량을 옆으로 눕히고 싶어서 옆 방향 토크를 줬는데,
동시에 위쪽 방향으로 남는 회전 효과가 있어서
순수하게 옆으로 넘어가지 않고 비스듬한 축을 중심으로 돈다.
```

이것이 B_eff 직관이다.

---

# 16. B_eff는 실제 자기장인가?

중요한 질문이다.

**B_eff는 실험실에 실제로 존재하는 하나의 정자기장이 아니다.**

B_eff는 회전좌표계에서 운동을 단순하게 표현하기 위한 **유효 자기장**이다.

실제로 존재하는 것은,

```text
B0: z축 방향의 강한 정자기장
B1: xy 평면에서 RF 주파수로 회전하는 약한 자기장
```

이다.

그런데 회전좌표계로 들어가면, 좌표계 자체가 돌고 있기 때문에 관성력 비슷한 항이 생긴다.

그 결과 운동방정식을 다음처럼 단순화할 수 있다.

```text
dM/dt = γ M × B_eff
```

즉 회전좌표계 안에서는 M이 마치 B_eff라는 하나의 자기장 둘레로 세차운동하는 것처럼 보인다.

따라서 B_eff는,

```text
실제 B1 + 회전좌표계 때문에 생기는 offset 효과
```

를 합친 개념이다.

---

# 17. Bloch equation으로 보면

실험실 좌표계에서 Bloch equation은 대략,

```text
dM/dt = γ M × B
```

이다.

여기서 B는 전체 자기장이다.

```text
B = B0 z + B1(t)
```

B1(t)는 xy 평면에서 RF 주파수로 회전한다.

이제 RF와 함께 도는 회전좌표계로 변환하면, 운동방정식은 다음처럼 바뀐다.

```text
(dM/dt)_rot = γ M × B_eff
```

여기서,

```text
B_eff = B1 x′ + (B0 - ωRF/γ) z
```

같은 형태가 된다.

이 식은 부호 convention에 따라 조금씩 달라진다.

만약 기준을 라모어 주파수와 RF 주파수의 차이로 쓰면,

```text
B_eff,z ∝ Δω / γ
```

가 된다.

첨부 파일에서는 `Δω = ωRF - ω0`에 해당하는 방식으로,

```text
B_eff = B1 x′ + (Δω/γ) z
```

를 사용한다. 

중요한 것은 부호보다 구조다.

```text
offset이 있으면 B_eff에 z성분이 생긴다.
```

---

# 18. 공명일 때 M의 움직임

공명 조건:

```text
Δf = 0
```

그러면,

```text
B_eff = B1 x′
```

처음 M은 +z 방향이다.

M은 B_eff, 즉 x′축 둘레로 회전한다.

따라서 시간에 따라,

```text
M_z = cos θ
M_y′ = sin θ 또는 -sin θ
M_x′ = 0
```

같은 형태로 변한다. 부호는 B1 방향과 γ 부호 convention에 따라 달라질 수 있다.

핵심은,

```text
M의 크기는 유지되고,
방향만 z에서 xy 평면으로 회전한다.
```

90° pulse 후에는,

```text
M_z ≈ 0
M_xy ≈ M0
```

가 된다.

그래서 transverse magnetization이 최대가 된다.

---

# 19. offset이 있을 때 M의 움직임

offset이 있으면,

```text
B_eff = B1 x′ + offset z
```

이다.

처음 M은 +z 방향이다.

이때 M은 B_eff 둘레로 돈다.

문제는 B_eff 자체가 z성분을 갖고 있기 때문에, M과 B_eff가 처음부터 어느 정도 평행 성분을 가진다는 것이다.

벡터가 어떤 축 둘레로 돌 때, 그 축 방향 성분은 보존된다.

즉,

```text
M의 B_eff 방향 성분은 그대로 남는다.
```

그러면 M 전체가 완전히 xy 평면으로 넘어가기 어렵다.

그래서 offset이 커질수록,

```text
90° pulse를 줬다고 생각했는데 실제로는 90°가 안 됨
180° pulse를 줬다고 생각했는데 완전히 뒤집히지 않음
```

같은 일이 생긴다.

---

# 20. 왜 off-resonance에서는 flip이 덜 되나

처음 M은 +z 방향이다.

B_eff가 x′ 방향이면 M은 B_eff와 수직이다.

```text
M ⟂ B_eff
```

수직이면 회전 효과가 최대다.

하지만 B_eff가 z축 쪽으로 기울어지면, M은 B_eff와 완전히 수직이 아니다.

```text
M의 일부는 B_eff와 평행
M의 일부만 B_eff에 수직
```

평행한 성분은 회전하지 않는다.

수직 성분만 돈다.

따라서 M을 xy 평면으로 눕히는 능력이 떨어진다.

이게 off-resonance excitation이 약해지는 가장 중요한 직관이다.

---

# 21. 아주 중요한 세 가지 상황

## 21-1. Δf = 0

```text
B_eff = B1 x′
```

결과:

```text
정확한 flip
90° pulse → Mxy 최대
180° pulse → inversion 최대
```

---

## 21-2. |Δf|가 f1보다 작음

```text
|Δf| < f1
```

결과:

```text
B_eff가 조금만 기울어짐
flip이 약간 왜곡됨
phase도 약간 틀어짐
대체로 excitation은 잘 됨
```

---

## 21-3. |Δf|가 f1보다 큼

```text
|Δf| > f1
```

결과:

```text
B_eff가 z축 쪽으로 많이 기울어짐
M이 xy 평면으로 잘 안 넘어감
RF pulse가 잘 안 먹힘
```

이게 RF pulse bandwidth와 연결된다.

---

# 22. RF bandwidth와 offset

RF pulse는 모든 주파수의 스핀을 똑같이 excitation하지 않는다.

중심 주파수에 가까운 스핀은 잘 excitation되고, 멀리 떨어진 스핀은 덜 excitation된다.

즉 RF pulse는 주파수 선택성을 가진다.

```text
Δf가 작다 → 잘 뒤집힘
Δf가 크다 → 덜 뒤집힘
```

B1이 강하고 pulse가 짧으면 bandwidth가 넓다.

B1이 약하고 pulse가 길면 bandwidth가 좁다.

직관적으로,

```text
강하게 짧게 밀면 약간 박자가 안 맞아도 넘어감
약하게 오래 밀면 정확히 박자가 맞아야 누적됨
```

이다.

---

# 23. slice selection과 B_eff

MRI에서 slice selection은 RF offset 개념을 적극적으로 이용한다.

z방향으로 gradient Gz를 걸면, 위치에 따라 B0가 달라진다.

```text
B(z) = B0 + Gz z
```

그러면 위치에 따라 라모어 주파수도 달라진다.

```text
f0(z) = γ̄ [B0 + Gz z]
```

RF pulse의 중심 주파수를 fRF로 정하면, 그 주파수와 맞는 위치의 스핀만 공명한다.

즉,

```text
fRF = f0(z0)
```

인 위치 z0가 선택된다.

z0에서 멀어질수록,

```text
Δf(z) = fRF - f0(z)
```

가 커진다.

그러면 B_eff의 z성분이 커져서 excitation이 약해진다.

결국,

```text
RF pulse + slice selection gradient
→ 특정 위치만 Δf ≈ 0
→ 그 slice만 flip됨
```

이 된다.

이것이 slice selection의 기본 원리다.

---

# 24. chemical shift도 offset이다

물 분자 속 proton과 지방 분자 속 proton은 같은 B0 안에 있어도 전자구름 환경이 달라서 실제로 느끼는 자기장이 약간 다르다.

그래서 라모어 주파수가 약간 다르다.

이것이 chemical shift다.

예를 들어 fat과 water는 3.5 ppm 정도 차이가 난다.

3T에서는 대략 440 Hz 정도 차이가 된다.

즉 water에 맞춘 RF를 쏘면 fat proton 입장에서는 offset이 있다.

```text
Δf_fat ≠ 0
```

그래서 fat과 water는 RF pulse에 대해 서로 다른 위상과 excitation을 보일 수 있다.

Dixon imaging, fat suppression, chemical shift artifact 등이 여기와 연결된다.

---

# 25. B0 inhomogeneity도 offset이다

B0가 공간적으로 완벽히 균일하지 않으면, 위치마다 f0가 조금씩 달라진다.

RF는 하나의 주파수로 들어오는데, 각 위치의 f0가 다르면 위치마다 offset이 생긴다.

```text
Δf(r) = fRF - f0(r)
```

따라서 어떤 위치에서는 RF가 공명이고, 어떤 위치에서는 off-resonance다.

결과적으로,

```text
flip angle 불균일
신호 저하
phase error
fat suppression 실패
banding artifact
```

등이 생길 수 있다.

---

# 26. B1 inhomogeneity와 offset의 차이

헷갈리기 쉬운 것이 B1 inhomogeneity와 RF offset이다.

둘은 다르다.

## B1 inhomogeneity

```text
B1 세기가 위치마다 다름
→ f1 = γB1이 달라짐
→ 같은 pulse 시간에도 flip angle이 달라짐
```

즉 문제는 **횡방향 회전시키는 힘의 크기**가 다른 것이다.

---

## RF offset / off-resonance

```text
RF 주파수와 라모어 주파수가 다름
→ B_eff에 z성분이 생김
→ 회전축 자체가 기울어짐
```

즉 문제는 **회전축 방향**이 달라지는 것이다.

물론 실제 MRI에서는 둘이 동시에 존재한다.

```text
B1도 위치마다 다르고,
B0도 위치마다 달라서 offset도 생긴다.
```

그래서 실제 flip map은 복잡해진다.

---

# 27. B_eff와 adiabatic pulse

심화로 가면 B_eff는 adiabatic pulse 이해의 핵심이다.

일반적인 hard pulse에서는 B_eff가 대체로 고정되어 있고, M이 그 둘레로 돈다.

하지만 adiabatic pulse에서는 RF amplitude와 frequency를 천천히 바꾼다.

즉,

```text
B1(t)가 변하고
Δf(t)도 변한다
```

그러면 B_eff의 방향이 시간에 따라 천천히 움직인다.

adiabatic 조건이 만족되면 M은 B_eff를 따라간다.

즉,

```text
M이 B_eff 주변에서 빠르게 돌면서,
B_eff 방향이 천천히 변하면,
M은 그 축을 따라 끌려간다.
```

이 원리를 이용하면 B1 불균일이나 offset에 더 강한 inversion pulse를 만들 수 있다.

대표적으로,

```text
adiabatic inversion pulse
STIR
spectral-spatial pulse
BIR pulse
```

같은 것들이 연결된다.

---

# 28. rotating frame에서 “정지한 B1”이 정말 가능한가

실제 RF field는 보통 선형 편광 field일 수 있다.

선형 RF field는 수학적으로 두 개의 반대 방향 회전 성분으로 분해할 수 있다.

```text
linear B1 = clockwise rotating component + counterclockwise rotating component
```

양성자의 라모어 회전 방향과 같은 성분은 공명해서 효과가 크다.

반대 방향 성분은 너무 빠르게 상대 회전하므로 평균적으로 거의 효과가 없다.

이것을 rotating wave approximation, RWA라고 한다.

따라서 회전좌표계에서는 유효하게 한쪽 방향으로 도는 B1 성분만 남겨서,

```text
B1이 x′ 방향에 고정되어 있다
```

고 해석한다.

고급으로 갈수록 여기서 B1/2 factor나 circularly polarized B1 표현이 중요해질 수 있다. 하지만 기본 MRI 설명에서는 보통 유효 B1을 이미 그 기준에 맞춰 쓴다.

---

# 29. RF offset의 부호가 실제로 바꾸는 것

Δf의 부호는 B_eff의 z성분 방향을 바꾼다.

```text
Δf > 0 → B_eff가 +z 쪽으로 기울어짐
Δf < 0 → B_eff가 -z 쪽으로 기울어짐
```

부호가 바뀌면 최종 M의 방향과 phase가 달라진다.

다만 excitation magnitude만 보는 경우, 즉 Mxy의 크기만 보면 ±offset이 대칭적으로 보일 때가 많다.

하지만 phase까지 보면 다르다.

MRI 신호는 크기뿐 아니라 phase도 중요하기 때문에, offset의 부호는 실제 영상에서 의미가 있다.

예를 들어,

```text
off-resonance phase accrual
chemical shift phase
Dixon in-phase / opposed-phase
bSSFP banding
```

등에서는 부호와 phase가 중요하다.

---

# 30. B_eff 관점에서 보는 90° pulse 실패

목표가 90° pulse라고 하자.

공명에서는,

```text
M: +z → xy plane
```

으로 간다.

그런데 offset이 있으면 M은 B_eff 둘레로 돈다.

이때 두 가지 문제가 생긴다.

첫째, 회전축이 x′가 아니다.

```text
B_eff가 z축 성분을 가짐
```

둘째, 회전속도도 B1 기준이 아니라 B_eff 기준이다.

```text
f_eff = sqrt(f1² + Δf²)
```

그래서 pulse 시간을 공명 기준으로 90°에 맞춰도, 실제로는 B_eff 둘레 회전각이 약간 다르다.

결과적으로,

```text
Mxy 크기도 예상과 다르고,
Mxy phase도 예상과 다르다.
```

---

# 31. B_eff 관점에서 보는 180° pulse 실패

180° pulse는 M을 +z에서 -z로 뒤집어야 한다.

공명에서는 x′축 둘레로 180° 회전하면 된다.

하지만 offset이 있으면 B_eff가 z축 쪽으로 기울어져 있다.

이때 M의 B_eff 방향 성분은 보존된다.

그러므로 완벽하게 -z로 가지 못한다.

즉 off-resonance에서는 inversion 효율이 떨어진다.

이것이 inversion recovery, fat suppression, selective saturation에서 매우 중요하다.

특히 fat suppression pulse는 fat만 90° 또는 180°로 잘 처리해야 하는데, B0 inhomogeneity가 있으면 fat의 Δf가 위치마다 달라져 suppression이 실패할 수 있다.

---

# 32. B_eff와 phase

M이 xy 평면에 생기면, 그 방향이 phase다.

공명에서 x′축을 기준으로 정확히 90° pulse를 주면 M은 예측 가능한 y′ 방향으로 생긴다.

하지만 offset이 있으면,

```text
B_eff가 기울어짐
B1이 회전좌표계에서 drift함
M이 추가적인 z축 precession을 함
```

이 모두 합쳐져서 Mxy phase가 달라진다.

따라서 off-resonance는 단순히 신호 크기만 낮추는 것이 아니라 phase error도 만든다.

MRI에서는 phase error가 영상 위치, 주파수 인코딩, Dixon, EPI distortion 등에 큰 영향을 준다.

---

# 33. B_eff와 relaxation

지금까지는 relaxation을 무시했다.

즉 M의 길이가 보존된다고 봤다.

하지만 실제로는 T1, T2 relaxation이 있다.

Bloch equation을 더完整하게 쓰면,

```text
dMx/dt = γ(M × B)x - Mx/T2
dMy/dt = γ(M × B)y - My/T2
dMz/dt = γ(M × B)z - (Mz - M0)/T1
```

RF pulse가 매우 짧으면 relaxation을 무시해도 된다.

하지만 긴 RF pulse, saturation pulse, adiabatic pulse에서는 relaxation도 중요해진다.

그래도 RF offset과 B_eff의 기본 구조는 그대로다.

```text
회전축은 B_eff
그 위에 relaxation이 추가됨
```

이라고 보면 된다.

---

# 34. 직관을 하나로 묶으면

아주 간단히 정리하면 다음과 같다.

```text
B0
→ 양성자를 z축 주변으로 돌게 하는 큰 자기장

f0
→ 그 회전 주파수, 라모어 주파수

RF B1
→ 양성자 회전에 맞춰 같이 도는 약한 횡자기장

RF offset Δf
→ RF 주파수와 양성자 라모어 주파수의 차이

Δf = 0
→ 회전좌표계에서 B1이 정지
→ M은 B1 축 둘레로 깨끗하게 flip

Δf ≠ 0
→ 회전좌표계에서 z축 방향의 offset 성분이 생김
→ M은 B1이 아니라 B_eff 둘레로 회전

B_eff
→ B1 횡성분과 offset z성분의 합성 유효장
```

---

# 35. 한 문장 버전

가장 중요한 한 문장으로 압축하면 이렇다.

> **RF offset은 “RF가 양성자와 박자가 얼마나 어긋났는가”이고, 그 어긋남은 회전좌표계에서 z축 방향의 유효 자기장처럼 작용한다. 그래서 순자화 M은 B1만 보는 것이 아니라 B1과 offset z성분이 합쳐진 B_eff 둘레로 회전한다.**

---

# 36. 수식 최소 세트

MRI RF offset과 B_eff를 이해하는 데 필요한 최소 수식은 이것들이다.

```text
f0 = γ̄ B0
```

```text
Δf = fRF - f0
```

```text
f1 = γ̄ B1
```

```text
B_eff = B1 x′ + (Δω/γ) z
```

```text
f_eff = sqrt(f1² + Δf²)
```

```text
tan α = Δf / f1
```

```text
θ_resonant = 2π f1 t
```

여기서,

```text
α = B_eff가 x′축에서 z축 쪽으로 기울어진 각도
θ = 공명이라고 가정했을 때의 flip angle
```

이다.

# 37. 매우 쉬운 최종 비유

RF pulse를 “박자 맞춰 미는 손”이라고 하자.

```text
양성자 = 빠르게 도는 팽이
B0 = 팽이가 돌게 만드는 수직 자기장
RF B1 = 옆에서 박자 맞춰 미는 손
Δf = 손의 박자가 팽이 회전보다 빠르거나 느린 정도
B_eff = 팽이 입장에서 느끼는 실제 밀림 방향
```

박자가 정확히 맞으면,

```text
손이 계속 같은 방향에서 밀어준다.
→ 팽이가 잘 기울어진다.
```

박자가 어긋나면,

```text
손의 방향이 조금씩 미끄러진다.
→ 옆으로만 미는 게 아니라 위아래 회전 효과가 섞인다.
→ 팽이는 원하는 방향으로 깔끔하게 눕지 않는다.
```

그 “섞인 실제 밀림 방향”이 바로 B_eff다.
